DataLayer Relay Test Page

Test your server-side tagging relay implementation

How to Use This Test Page

  1. Open your browser's Developer Console (F12 or right-click > Inspect > Console tab)
  2. Click any test button below to fire a dataLayer event
  3. Watch the Event Log on the right to see what data was pushed
  4. Check the console for detailed output and relay activity
  5. Verify events are being sent to your server-side GTM container
Note: This test page uses the pre-configured relay script (relay-configured.js) pointing to https://localhost:8888. Make sure your Docker GTM containers are running before testing.
